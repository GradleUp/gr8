fun systemClassesToolchain(compiler: JavaCompiler)

The java compiler toolchain to use for discovering system classes when running R8

The system classes from this Java toolchain will be passed to R8 when invoking it, enabling you to target any version of Java toolchain while building on newer JDKs. Defaults to the toolchain used to compile.

Usage:

systemClassesToolchain(javaToolchains.compilerFor {
  languageVersion.set(JavaLanguageVersion.of(11))
})
